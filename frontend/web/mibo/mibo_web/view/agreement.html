1.2 用户必须对自己所注册账号下的一切行为负责，包括直播的视频内容、文字内容、语音内容，为之承担所有的法律责任。用户必须保证在蜜播APP上所制作或传播的信息的合法性，同时保证相关信息不侵犯任何第三方知识产权，若由于用户未遵守协议内容而产生的任何后果由用户承担。用户不得制作和传播包含以下信息的内容：

1)反对宪法所确定的的基本原则的；

2)危害国家安全，泄露国家机密，颠覆国家政权，破坏国家统一的；

3)损害国家荣誉和利益的；

4)煽动民族仇恨、民族歧视，破坏民族团结的；

5)破坏国家宗教政策，宣扬邪教和封建迷信的；

6)散步谣言，扰乱社会秩序，破坏社会稳定的；

7)散步淫秽、色情、粗俗、猥亵、赌博、暴力、凶杀、恐怖、教唆犯罪或其它道德上令人反感的内容；

8)侮辱或者诽谤他人，侵害他人合法权益的；

9)含有法律、行政法规、规章、条例以及任何具有法律效力之规范所限制或禁止的其它内容的。进行其它任何违法以及侵犯其它个人、公司、社会团体、组织的合法权益的行为。

2.隐私保护

2.1 请勿在蜜播平台上向其他用户泄露自己的重要信息，包括自己的详细地址、财产账户、支付密码等，否则由此带来的任何损失由用户自行承担。

2.2 蜜播承诺不会将用户在使用蜜播过程中向蜜播平台提交的任何个人信息（包括姓名、性别、年龄、身份证号、住址、电话、银行账户等个人资料）未经用户同意就向第三方公开，以下情况除外：

1)依据相关法律法规要求向政府、司法机关提供您的个人信息；

2)由于用户个人原因（如共享账户等）导致的个人信泄露；

3)由于不可抗力导致的信息泄露。

3.1 用户因电信部门的通讯线路故障、网络等原因遭受的损失，蜜播不承担责任。
